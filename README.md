# BMGIS Policy Pages

This folder contains public policy pages for the BMGIS Android app.

Suggested GitHub repository name:

- `bmgis-policy`

Suggested repository description:

- `Public privacy policy and account deletion information for the BMGIS Android app.`

Suggested files to upload:

- `index.html` — Privacy Policy
- `deletion.html` — Account Deletion Information

Suggested GitHub Pages URL after publishing:

- `https://hashnol.github.io/bmgis-policy/`

After upload:

1. Create a new public GitHub repository.
2. Upload `index.html`, `deletion.html`, and optionally this `README.md`.
3. Go to Settings → Pages.
4. Set Source to `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait for the Pages URL to appear.

Before publishing, replace:

- `bmgis.my@gmail.com`

with your real support contact email.
